Grid 속성 설명:

  • display: grid - 그리드 컨테이너 설정
  • grid-template-columns - 열(column) 구조 정의
  • grid-template-areas - 영역 이름으로 레이아웃 구성
  • gap - 그리드 아이템 간격
  • fr 단위 - 사용 가능한 공간의 비율
  • repeat() - 반복되는 값을 간단히 표현
  • minmax() - 최소/최대 크기 설정
  • auto-fit - 반응형 그리드에 활용
